1.0 Konsep Asas Kod Arahan

Apa itu Pemboleh Ubah?

Pemboleh ubah (Variable) adalah tempat untuk menyimpan data dalam memori komputer. Contohnya, menyimpan markah murid.

Jenis Data Utama:

  • Integer: Nombor bulat (Contoh: 10, 500)
  • Float: Nombor perpuluhan (Contoh: 3.14, 5.5)
  • String: Teks (Contoh: "Hello Dunia")

Contoh Kod Python

Berikut adalah contoh penggunaan syarat if-else:

markah = 75 if markah >= 40: print("Lulus") else: print("Gagal")

Output bagi kod di atas ialah: Lulus
